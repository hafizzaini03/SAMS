Project Overview

The Smart Air-Conditioner Monitoring System (SaMS) is a non-intrusive
energy monitoring solution designed for institutional environments. By utilizing
current sensing technology, SaMS tracks the operational state of AC units in
real-time, helping facilities reduce energy waste and optimize maintenance
schedules.

Hardware Architecture

SCT-013-000 Sensor: Non-invasive split-core current transformer for safe
current measurement.

ESP32 Microcontroller: Dual-core processor handling signal sampling,
processing, and Wi-Fi connectivity.

Signal Conditioning: Precision burden resistor and DC bias circuit to
interface AC signals with DC ADC pins.

Key Features

Non-Intrusive Installation: No need to modify primary electrical wiring,
ensuring safety and compliance.

State Detection: Advanced persistence logic to accurately differentiate
between ON, OFF, and Standby modes.

Centralized Monitoring: Data is transmitted to a centralized dashboard for
multi-room oversight.
